# Lab 09 — Angular Components (COMP 3133)

**Student ID:** 101443133

Angular lab exercises: manual `StudentsComponent`, CLI-generated `StudentComponent`, and template binding with ES2015 template literals.

## Project layout

- `student-app/` — Angular application (`ng new student-app`)

## Run locally

```bash
cd student-app
npm install
npm start
```

Open `http://localhost:4200`.

## Build

```bash
cd student-app
npm run build
```

## GitHub

1. Create a new empty repository on GitHub (no README if you already have this one).
2. From this folder:

```bash
git remote add origin https://github.com/<your-username>/<your-repo>.git
git branch -M main
git push -u origin main
```

Replace `<your-username>` and `<your-repo>` with your account and repository name.
